# Hospital Management System
Python | Django | MySQL

A web-based Hospital Management System developed using Python (Django framework) and MySQL to manage doctors, patients, and appointments with role-based access control.

## Project Features
- Admin, Doctor, and Patient modules
- Online appointment booking and management
- Unique patient ID generation
- Advanced search and reporting
- Secure authentication and authorization
- Responsive user interface

## Technologies Used
- Language: Python
- Framework: Django
- Database: MySQL
- Frontend: HTML, CSS, JavaScript, jQuery, AJAX
- IDE: PyCharm
- Server: XAMPP (MySQL)
- Browser: Google Chrome, Mozilla Firefox

## How to Run the Project

### Step 1: Download and Extract
- Download the ZIP file
- Extract the file
- Copy the `hms` folder and paste it on the Desktop

### Step 2: Database Setup
- Start XAMPP and run MySQL
- Create a database named `hmspythondb`
- Import the SQL file from the SQL File folder
